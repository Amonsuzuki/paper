## 13. Notes for Copilot Usage

### When to use `[COPILOT: HELP]`
Use this tag when you want Copilot to:
- Generate content
- Create new files
- Perform analysis
- Answer specific questions
- Complete structured tasks (like 60qa)

### When to use `[COPILOT: REVIEW]`
Use this tag when you want Copilot to:
- Review existing content
- Provide feedback
- Suggest improvements
- Identify issues

### When to use `[COPILOT: NO ACTION]`
Use this tag for:
- Personal notes
- Reference information
- Ideas under consideration
- Organizational structures

### When to use `[DONE]`
Mark tasks as `[DONE]` when:
- The task is completed
- No further action is needed
- You want to keep it for reference
