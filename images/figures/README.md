# Figures Directory

This directory contains figures from research papers and diagrams for documentation.

## Contents

Place your paper figures and diagrams here with descriptive names.

### Naming Convention

Use descriptive names with hyphens:
- `model-architecture.png`
- `dft-energy-functional.png`
- `training-workflow.svg`

### Example Usage in Markdown

```markdown
<p align="center">
  <img src="../images/figures/your-figure.png" alt="Description" width="600">
  <br>
  <em>Figure 1: Caption describing your figure</em>
</p>
```

## Tips

- Use PNG for diagrams with transparency
- Use SVG for scalable vector graphics (recommended)
- Use JPEG for photographs
- Optimize file size before committing
- Add descriptive alt text for accessibility
